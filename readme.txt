custom code widget help to add Short Code, PHP Code, HTML Code and Simple Text in Sidebar.

== Description ==

Custom Code Widget is simple widget allow you to insert any code Text/HTML and run PHP Code or Short Code. This Widget parse PHP code into simple text and much more.

== Frequently Asked Questions ==


= Can we run PHP Code in this widget ?=

  Yes, All PHP code must be enclosed in the standard php opening and closing tags ( `<?php` and `?>` ) for it to be recognized and executed.
  
= There's some kind of error  class-custom-code-widget.php(188) : eval()'d code on line 6 =

  That  means  your PHP code is incorrect or otherwise broken.

= Can We add Simple HTML Markup ?=

  just simple add html markup which you want display in widget.

= Yes, Simple Text also Work in Custom Code Widget ?=

  As our normal routine we insert any text in widget.
